람다식(Lambda)
=========
- 익명 메소드를 지칭하는 용어
- 간결한 방법으로 함수를 묘사하기 위함
- 모든 것이 함수로 이루어져 있음
- 람다 선언 연산자 =>를 사용하여 람다의 매개 변수 목록을 구분

```cs
(input-parameters) => expression
```
