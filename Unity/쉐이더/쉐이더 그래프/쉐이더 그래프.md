Shader Graph에 대하여
===========



### 2D Render -> Sprite Lit Graph
- 빛에 반응하는 그래프

### 2D Render -> Sprite Unlit Graph
- 빛에 반응하지 않는 그래프

### Unlit Graph
- 3D에서 빛에 반응하지 않는 그래프

### PBR Graph
- Physically Base Rendering(물리 기반 렌더링)
- 현실 세계에서 눈에 보이는 것과 똑같이 나타내는 그래프

### VFX Shader Graph
- 파티클을 많이 만들 때 사용하는 그래프

### Sub Graph
- 그래프를 묶는 것
