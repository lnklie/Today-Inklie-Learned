"(쌍따옴표) 와 \(역슬래쉬) 출력하기
-----------
- \를 앞에 붙여주면 된다!

```c++
#include <iostream>
using namespace std;

int main(void)
{
	cout << "         ,r\'\"7" << endl;
	cout << "r`-_   ,'  ,/" << endl;
	cout << " \\. \". L_r'" << endl;
	cout << "   `~\\/" << endl;
	cout << "      |" << endl;
	cout << "      |" << endl;
}
```
